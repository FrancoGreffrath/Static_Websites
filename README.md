Online Shop – Static Website

This project is a static online shopping webpage built using HTML, CSS, and Bootstrap. It showcases products in a responsive grid layout and demonstrates the use of Bootstrap’s grid system, components, and styling utilities.

The site is fully front-end only and does not include backend functionality or real purchasing logic.

Features

Responsive Design using Bootstrap’s grid system

Product layout displayed in rows and columns

Responsive product images using img-fluid

Product cards with name, price, and Buy button

Footer section with a horizontal contact form

Simple hover effects using custom CSS

Technologies Used

HTML5

CSS3

Bootstrap 5 (via CDN)
